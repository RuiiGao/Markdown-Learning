## 2.使用Github作为图床
- **方法一**：使用相对路径（实际上同上）：图片和 `.md` 文件在同一个仓库；无论仓库被克隆到哪里，图片链接都不会失效。
  - 直接上传图片，md中写相对路径
  - 注意：`./` 是本级； `../` 是上级
- **方法二**：使用 Github 提供的链接 = **Github作为图床**
  - 如果你想在**外部**的 Markdown 文件（比如在其他网站、博客或另一个仓库的 issue ）中使用这张图片，你需要一个可以公开访问的URL。Github为仓库的每个文件都提供了这样的 "Raw" 链接。
  - 步骤
    1. 上传图片
